A collection of classes converting byte[] into {@link java.lang.String}s and vice-versa.

This package supports few standard algorithms like {@linkplain org.usrz.libs.crypto.codecs.HexCodec HEX}, {@linkplain org.usrz.libs.crypto.codecs.Base32Codec Base32} and {@linkplain org.usrz.libs.crypto.codecs.Base64Codec Base64}.
